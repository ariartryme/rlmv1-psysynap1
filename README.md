# Revarie1 – Study Portal

Production branch for `research.imace.online`.

## Tech Stack
- Next.js 16
- Tailwind CSS 4
- Cloudflare D1 (via Worker API)

## Environment Variables
Set in Vercel: `VAULT_API_URL`, `VAULT_API_KEY`, `NEXT_PUBLIC_CDN_URL`

## Deployment
Deployed to Vercel from the `Production` branch.
